# 📊 AI Customer Insights System (n8n)
![Workflow](https://github.com/josephautomationke/ai-customer-insights-n8n/blob/main/Ai%20Customer%20insights%20p.png?raw=true)
## 🧠 Problem
Businesses receive many customer messages but struggle to extract useful insights like common complaints, needs, and trends.

---

## 💡 Solution
I built an AI-powered workflow using n8n that analyzes customer messages and extracts meaningful insights automatically.

---

## ⚙️ How It Works
1. Customer messages are collected (Google Sheets / webhook)
2. Data is processed in n8n
3. Messages are sent to an AI model
4. AI analyzes and extracts:
   - Customer needs
   - Complaints
   - Trends
5. Results are stored for business decision-making

---

## 🛠️ Tech Stack
- n8n (automation)
- OpenAI API
- Google Sheets
- JavaScript
